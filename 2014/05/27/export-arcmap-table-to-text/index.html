Export ArcMap table to text

Posted on 2014-May-27 by matt

First of all, is it really text you’re after? There’s:

ArcToolbox > Conversion Toools > Table to Excel

Copy paste to Arcmap Location bar:
Toolboxes\System Toolboxes\Conversion Tools.tbx\Excel

Also, once you know the secret it can be easier to just copy > paste:

  1. Open attribute table (Ctrl-2x-click layer in Table of Contents)
  2. Select All (Ctrl-A)
  3. R-click on any row heading > Copy selected

Yes I really want text

Then you’re looking for the cleverly hidden “Export Feature Attribute to ASCII (Spatial Statistics)”

Toolboxes\System Toolboxes\Spatial Statistics Tools.tbx\Utilities

(Thank you Mike Richards!)
